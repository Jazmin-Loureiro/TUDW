Elementos de navegacion y orientacion del usuario

Los elementos de navegacion y orientacion tienen como funcion basica informar constantemente al usuario acerca de donde se encuentra.

Las dos formas basicas para no perder al usuario son:
Que no se aburra
que no se "pierda" navegando

Coherencia del disenio

La primera regla para indicarle a nuestro usuario que sigue estando en nuestra Web es manteniendo una choerencia de disenio, es decia, una uniformidad en la estructura de las paginas que forman nuestro stio, y tambien en lo colores empleados
