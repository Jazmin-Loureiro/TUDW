INTRODUCCION HTML


Este es un curso que pretende introducior al lector en el mundo HTML. No se requieren conocimientos previos, se empezara desde cero. No hay que saber nada de programacion para utilizar HTML, basta aprender un par de cosas sencillas para empezar a crear paginas.

Es MUY FACIL aprender HTML. Tan solo necesitas un editor de texto, un curso de html y un navegador web (como netscape o internet explore), ni siquiera necesitas estar conectado a Internet.
